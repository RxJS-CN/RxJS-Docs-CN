    TeardownLogic

    这个接口描述了 Observable 构造函数和静态方法 create 接收的函数的返回对象。这个接口 的值可以用来取消对当前 Observable 的订阅。

    TeardownLogic 可以是:

    函数

    该函数不接收参数。 当创建的 Observable 的消费者调用 unsubscribe时，该函数被调用。

    AnonymousSubscription

    AnonymousSubscription 是一个拥有 unsubscribe 方法的简单对象。 该方法和上面函数行为一致。

    void

    如果创建的 Observable 不需要清理任何资源，该函数不需要返回任何值。
